# Epilepsy Analysis: A Multidimensional Data Study

## 📌 Project Overview
This project analyzes a public epilepsy dataset to uncover patterns in seizure frequency, lifestyle factors, medication adherence, and demographic differences. The goal is to support more personalized treatment strategies and improve patient outcomes.

## 🧠 Key Questions
- How do seizure patterns vary over time?
- What lifestyle factors influence seizure frequency?
- How does medication adherence impact outcomes?
- Are there demographic differences in seizure behavior?

## 📂 Dataset
Source: Public epilepsy datasets from Kaggle/NIH  
Includes:
- Seizure timestamps
- Seizure type & duration
- Medication usage
- Sleep & stress levels
- Demographics

## 🧹 Data Cleaning
- Removed duplicates
- Standardized date/time formats
- Addressed missing values
- Engineered new variables:
  - Seizures per week
  - Sleep deviation
  - Medication adherence score
  - Stress category

## 📊 Analysis Methods
- Descriptive statistics
- Comparative analysis across groups
- Correlation analysis (sleep, stress, adherence)
- Time-series trend analysis

## 📈 Visualizations
- Time-series charts
- Correlation heatmaps
- Scatter plots
- Demographic bar charts

## 🧩 Key Findings
- Sleep deviation and stress strongly correlate with seizure frequency
- Medication adherence is a major predictor of seizure reduction
- Demographic groups show distinct seizure patterns

## 🚀 Recommendations
- Improve medication adherence through reminders and simplified schedules
- Encourage consistent sleep routines
- Use digital tracking tools (apps, wearables)
- Provide demographic-specific education
